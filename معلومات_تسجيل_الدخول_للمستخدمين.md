## ملاحظات إضافية

1. **يمكن تسجيل الدخول باستخدام:** اسم المستخدم أو البريد الإلكتروني
2. **لإنشاء هذه المستخدمين:** قم بتشغيل الأمر التالي:
   ```bash
   php artisan db:seed --class=Database\\Seeds\\CreateTestUsersSeeder
   ```
3. **المستخدم الإداري:** يتم إنشاؤه عادة أثناء عملية التثبيت
4. **جميع المستخدمين:** نشطين ومفعلين افتراضياً
