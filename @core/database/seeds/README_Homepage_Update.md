# تحديث محتوى الصفحة الرئيسية - صيانة تك

## نظرة عامة

تم إنشاء ملفات لتحديث محتوى الصفحة الرئيسية في قاعدة البيانات بناءً على ملخص موقع صيانة تك.

## الملفات المتوفرة

### 1. UpdateHomepageContentSeeder.php
Seeder شامل لتحديث محتوى الصفحة الرئيسية:
- تحديث عنوان الصفحة الرئيسية في جدول `pages`
- تحديث محتوى Page Builder (جميع الأقسام)
- تحديث Static Options (العنوان، الوصف، إلخ)

### 2. UpdateHomepageDatabase.sql
سكريبت SQL لتحديث قاعدة البيانات مباشرة:
- تحديث عنوان الصفحة الرئيسية
- تحديث Static Options
- تحديث صفحات About و Contact

## كيفية الاستخدام

### الطريقة الأولى: استخدام Seeder (موصى بها)

```bash
cd @core
php artisan db:seed --class=Database\\Seeds\\UpdateHomepageContentSeeder
```

### الطريقة الثانية: استخدام SQL Script

1. افتح قاعدة البيانات (phpMyAdmin أو أي أداة أخرى)
2. قم بتشغيل ملف `UpdateHomepageDatabase.sql`
3. تحقق من النتائج

### الطريقة الثالثة: التحديث اليدوي

1. انتقل إلى لوحة التحكم: `/admin/page-builder/home-page`
2. قم بتحديث محتوى Page Builder يدوياً
3. استخدم النصوص من ملف `ar.json` للترجمة

## المحتوى المحدث

### الصفحة الرئيسية
- **العنوان**: "صيانة تك - منصة خدمات الصيانة المنزلية والتقنية"
- **الوصف**: "منصة إلكترونية متكاملة لإدارة خدمات الصيانة المنزلية والتقنية. ربط العملاء بالفنيين المعتمدين مع تتبع مباشر للطلبات، أسعار شفافة، فواتير إلكترونية، وضمانات رقمية."

### الأقسام المحدثة
1. **Hero Banner**: العنوان والوصف الرئيسي
2. **الخدمات المتاحة**: عرض الخدمات
3. **لماذا صيانة تك؟**: المميزات
4. **فئات الخدمات**: الكهرباء، السباكة، التكييف، إلخ
5. **الإحصائيات**: تغطية المدن، فنيون معتمدون، خدمة 24/7
6. **كيف نعمل**: خطوات العمل
7. **أفضل الفنيين**: عرض الفنيين
8. **قصص العملاء**: شهادات العملاء
9. **المدونة**: أحدث المقالات

### الخدمات المتاحة
- كهرباء (Electrical)
- سباكة (Plumbing)
- تكييف (HVAC)
- أجهزة منزلية (Appliances)
- نجارة (Carpentry)
- دهان (Painting)

## ملاحظات مهمة

1. **Page Builder Content**: محتوى Page Builder يتم حفظه كـ serialized PHP arrays، لذلك يجب استخدام Seeder لتحديثه بشكل صحيح.

2. **الترجمة**: جميع النصوص الجديدة تم إضافتها إلى ملف `@core/resources/lang/ar.json`

3. **التحقق**: بعد التحديث، تحقق من:
   - الصفحة الرئيسية: `/`
   - لوحة التحكم: `/admin/page-builder/home-page`
   - Static Options: `/admin/general-settings/reading`

4. **النسخ الاحتياطي**: يُنصح بعمل نسخة احتياطية من قاعدة البيانات قبل التحديث.

## استكشاف الأخطاء

### المشكلة: Seeder لا يعثر على الصفحة الرئيسية
**الحل**: تأكد من تعيين الصفحة الرئيسية في `/admin/general-settings/reading`

### المشكلة: Page Builder لا يتم تحديثه
**الحل**: تحقق من أن الصفحة الرئيسية تستخدم Page Builder (تفعيل من `/admin/page-builder/home-page`)

### المشكلة: النصوص لا تظهر بالعربية
**الحل**: تأكد من أن اللغة العربية مفعلة وأن الموقع يعرض بالعربية

## الدعم

للمزيد من المعلومات، راجع:
- ملف الملخص: `ملخص_الموقع_الشامل.md`
- ملف اللغة: `@core/resources/lang/ar.json`
- لوحة التحكم: `/admin/page-builder/home-page`
