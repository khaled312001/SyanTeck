# تعليمات تحديث محتوى الصفحة الرئيسية

## خطوات سريعة

### 1. تحديث قاعدة البيانات باستخدام Seeder

افتح Terminal/Command Prompt في مجلد المشروع وقم بتنفيذ:

```bash
cd @core
php artisan db:seed --class=Database\\Seeds\\UpdateHomepageContentSeeder
```

### 2. التحقق من التحديث

1. افتح الموقع: `http://localhost/SyanTeck/`
2. تحقق من الصفحة الرئيسية
3. انتقل إلى لوحة التحكم: `/admin/page-builder/home-page`
4. راجع المحتوى المحدث

### 3. التحديث اليدوي (اختياري)

إذا كنت تريد تحديث محتوى معين يدوياً:

1. انتقل إلى: `/admin/page-builder/home-page`
2. اختر القسم الذي تريد تحديثه
3. قم بتعديل المحتوى
4. احفظ التغييرات

## ما تم تحديثه

✅ عنوان الصفحة الرئيسية  
✅ محتوى Page Builder (جميع الأقسام)  
✅ Static Options (العنوان، الوصف)  
✅ ملف اللغة العربية (ar.json)  
✅ صفحات About و Contact  

## ملاحظات

- تم إضافة جميع النصوص الجديدة إلى ملف اللغة العربية
- يمكن استخدام النصوص من `ar.json` في أي مكان في الموقع
- يُنصح بعمل نسخة احتياطية قبل التحديث

## الدعم

إذا واجهت أي مشاكل:
1. تحقق من أن الصفحة الرئيسية معينة في الإعدادات
2. تأكد من تفعيل Page Builder للصفحة الرئيسية
3. راجع ملف `README_Homepage_Update.md` للمزيد من التفاصيل
